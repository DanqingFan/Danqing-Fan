# 10 идеальных пешеходных маршрутов по Москве
Этот *8-километровый маршрут* – отличный способ познакомиться с главными улицами Москвы, и прогуляться по широким бульварам города. Начиная путь от **станции метро Арбатская**, стоит, прежде всего, пройти по **Старому** и **Новому Арбату** – с многочисленными барами, сувенирными магазинами, кафе и магазинами (обязательно загляните в ТЦ «Весна»!). После этого можно подняться по **Тверскому бульвару** до **пересечения с Тверской улицей**, и пойти дальше – до **Петровки**. Закончить маршрут можно на **Театральной площади**, и сходить в один из **ближайших музеев Кремля**.
